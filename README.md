## brunch starter based on the 0-starter
+ Sass
+ Auto Reload
+ ES6 with Babel
    - modules (require)
+ Bower package manager
    - lodash
    - jquery
+ jsx ready


### Starting a new project


brunch new `http://github.com/nashio/brunch-es6-starter` shiny-project

cd shiny-project

#### Develop
brunch watch --server

#### Production
brunch build --production
